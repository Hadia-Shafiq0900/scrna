# 🧬 Single-Cell RNA-seq Analysis (10X Dataset)

## 📌 Overview
This project demonstrates a complete workflow for preprocessing and basic analysis of **10X Genomics single-cell RNA sequencing (scRNA-seq)** data.  

The analysis follows established tutorials and uses Python-based tools (AnnData / Scanpy ecosystem) to process, explore, and visualize gene expression at single-cell resolution.

---

## 🎯 Purpose
The goal of this assignment is to:

- Understand preprocessing steps in scRNA-seq data  
- Learn how to handle 10X datasets  
- Perform quality control (QC), filtering, and normalization  
- Explore data using dimensionality reduction techniques  
- Visualize cell populations and gene expression patterns  
- Gain familiarity with AnnData structure and workflow
